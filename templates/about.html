About This Project

Architecture

This is a plugin-based FastAPI application that uses a hooks system (via pluggy) to modularly load features as "updates". The architecture allows new functionality to override or extend existing functionality using a priority-based system.

Key Features

  • Updates System: Auto-discovers modules from app/updates/ directory
  • Priority Override: Higher priority components replace lower priority ones
  • Async Database: SQLAlchemy 2.0 with aiosqlite
  • Redis Integration: Rate limiting and caching support
  • Docker Ready: Containerized deployment with docker-compose

Technology Stack

  • FastAPI - Web framework
  • SQLAlchemy 2.0 - ORM
  • Pluggy - Hook system
  • Redis - Caching and rate limiting
  • Pydantic - Data validation
  • Jinja2 - Template engine
